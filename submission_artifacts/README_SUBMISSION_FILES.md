# Submission Files

Upload these files to the corresponding fields:

- Design Decision Document: `DESIGN_DECISIONS.md`
- Requirements: `requirements.txt`
- Sample Output File: `SAMPLE_OUTPUT.json`
- Parse Quality Report: `PARSE_QUALITY_REPORT.csv`
- AI Usage Log, if requested separately: `AI_USAGE_LOG.md`

The sample output and parse quality row were generated from:

```bash
python cli.py --resume outputs/e2e_check/resume.pdf --jd outputs/e2e_check/jd.json
```
